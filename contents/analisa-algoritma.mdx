Binary Search merupakan algoritma yang sangat efisien karena memiliki Time Complexity logaritma dalam mencari sebuah nilai.
Binary search bekerja dengan cara memilih pivot yang merupakan nilai index yang berada di tengah dari sebuah list,
lalu membandingkan apakah nilai pivot lebih besar atau kurang dari nilai yang dicari,
Jika nilai tersebut lebih besar maka pivot akan berubah menjadi index yang berada di tengah antara nilai yang lebih besar dan nilai terakhir dari sebuah list.

### Questions

Given a sorted array of n integers and a target value, determine if the target exists in the array in logarithmic time using the binary search algorithm. If target exists in the array, print the index of it.

#### Example 1

Input : [2, 3, 5, 7, 9]  
Target : 7  
Output : 3

#### Example 2

Input : [2, 3, 5, 7, 9]  
Target : 4  
Output : -1
